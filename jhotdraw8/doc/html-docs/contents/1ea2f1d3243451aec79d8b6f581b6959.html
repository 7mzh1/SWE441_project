Rationale

UMLClass
JHotDraw 8 :: Meta Model :: Logical Model :: Rationale

Description

Describes a rationale for a design decision.

We create instances of the Rationale class inside of Decisions models, so that we can find them together with the DesignDecision objects.

We then link from a Rationale instance object to the DesignDecision object that we want to rationalize.
