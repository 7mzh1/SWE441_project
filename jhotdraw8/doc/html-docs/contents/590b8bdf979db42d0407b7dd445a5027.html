BiblioEntry

UMLClass
JHotDraw 8 :: Meta Model :: Logical Model :: BiblioEntry

Description

Class for bibliographic entries.

We create instances of the BiblioEntry class only in the Bibliography model, so that all bibliographic entries can be found in one place.

To reference a bibliographic entry, we can create a link from a model element or a diagram to a BiblioEntry instance.
