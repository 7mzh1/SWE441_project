public getPrevVertices(v: V): Iterable<V>

This graph traversal function takes a vertex and returns an iterable of the previous vertices.

An implementation of the traversal function will internally need some kind of knowledge about the underlying graph; for example by having a reference to the graph or by storing data about the graph internally.
