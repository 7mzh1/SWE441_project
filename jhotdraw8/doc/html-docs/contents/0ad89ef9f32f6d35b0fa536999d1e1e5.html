Description

Functional Interface for deciding whether an object was already visited.

The add() method in this interface has deliberately the same signature like the add() method in java.util.Set.

This functional interface allows to implement different strategies for remembering whether an object was already visited.

e.g. the add() method of a java.util.Set instance could be used. an attribute on the object v could be used to determine whether the object was already visited.
